# ApexOrbitTrails / ElytraTrail2
Bring Apex Legends-style rank orbits to Elytra glides. This page is written like a wiki so you can hand it to server owners or players as-is.

---

## English (EN)
Simple Description

This plugin brings the unique "Rank Orbit" system from the game Apex Legends into Minecraft.
With a highly customizable config file, you can create your own orbits. It supports all in-game Minecraft particles, and some can even have their colors specified using a hexadecimal (Hex) code.
You can also adjust the number of particles, animation speed, and whether the orbit activates only while gliding or whenever you’re airborne. Finally, set the item name for the icon to finish setup. After adding or modifying configurations, simply run /orbit reload to reload the config.

Note: The particle names differ from vanilla Minecraft. Please refer to this page:
https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Particle.html

The item names also differ from vanilla naming conventions. Please refer to this page:
https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/Material.html

To apply an orbit, open the menu with /orbit and click the corresponding icon — it will be equipped instantly.

For detailed instructions, please see the [Wiki](https://github.com/zishounekonanoda/MinecraftElytraOrbit/wiki)

簡単な説明
